Welcome to AngularBeans Pages.

AngularBeans is an effort to have one well integrated Web Framework for any Java-WEB-CDI container, it simplify the building of complex, realtime, JavaScript client side MV* , with javaEE/CDI backend applications, it generate AngularJs ready to use module containing injectables angular services (CDI beans proxies)

Features

- Bean validation support (converted to client side angularJS form validation)

- Built-in translation service

- SockJs support!

- Realtime Event broadcasting from backend to clients (as angular events)

- QueryModel: a query from the backend to front end('s) to update their model('s)

considering the following CDI bean:

HelloBean.java :

package helloWorld;

import angularBeans.api.AngularBean;
import angularBeans.api.NGReturn;
import angularBeans.realtime.RealTime;

import java.io.Serializable;

import javax.enterprise.context.RequestScoped;
import javax.ws.rs.GET;

@AngularBean
@RequestScoped
public class HelloBean implements Serializable {
	
	@GET
	public String sayHello(String name) {
		return "hello " + name;
	}
}

app.js :

'use strict';

angular.module('myModule', [ 'angularBeans' ]);

angular.module('myModule')
.controller('HelloCtrl', function($scope, helloBean) {


	$scope.sayHello = function(name) {
		helloBean.sayHello(name).then(function(result){
			$scope.result=result;	
		}
		);
	}

});

and the HTML5 file :

		{{result}}
	</div>
</body>
</html>
