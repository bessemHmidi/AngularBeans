Building AngularBeans: 


git clone git@github.com:bessemHmidi/AngularBeans.git
cd AngularBeans\angular-beans
mvn clean install

Maven dependency

<dependency>
<groupId>com.github.bessemHmidi</groupId>
<artifactId>angularBeans</artifactId>
<version>1.0.2-RELEASE</version>
</dependency>
